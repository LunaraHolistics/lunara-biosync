# 📸 Carrossel de Divulgação

## Como adicionar imagens

1. **Coloque suas imagens aqui**: Salve as imagens em formato **retrato** (vertical) nesta pasta
   - Formatos aceitos: `.jpg`, `.png`, `.webp`
   - Recomendado: Imagens em retrato (altura > largura)
   - Tamanho recomendado: ~800x1000px ou maior

2. **Nomeie as imagens sequencialmente**:
   ```
   imagem1.jpg
   imagem2.jpg
   imagem3.jpg
   imagem4.jpg
   imagem5.jpg
   ```

3. **Faça commit e push**:
   ```bash
   git add client/public/assets/carousel/
   git commit -m "Add carousel images"
   git push
   ```

4. **Pronto!** O carrossel será atualizado automaticamente no Vercel em 1-2 minutos

## Características do Carrossel

✅ **Autoplay automático** - Passa as imagens a cada 5 segundos
✅ **Responsivo** - Se adapta perfeitamente em mobile e desktop
✅ **Navegação manual** - Use os botões de seta para navegar
✅ **Indicadores** - Veja qual slide você está visualizando
✅ **Pausa/Reprodução** - Controle o autoplay com o botão ⏸/▶

## Localização na página

O carrossel aparece entre as seções:
- **Acima**: "O que é Bioressonância?"
- **Abaixo**: "Benefícios"

## Dicas

- Use imagens com boa qualidade e contraste
- Mantenha o aspecto retrato (3:4 ou 9:16)
- Teste em mobile para garantir que ficou bom
- Você pode adicionar quantas imagens quiser!
